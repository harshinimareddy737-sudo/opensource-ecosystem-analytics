# 🌐 Open Source Ecosystem Analytics Platform

A platform that analyzes the health and activity of the open source ecosystem using real GitHub event data.

## Dataset
- **Source:** GitHub Archive (gharchive.org)
- **Data:** January 1, 2025 (3PM – 5PM UTC)

## Libraries Used
- Polars, Pandas, NumPy, Matplotlib, Streamlit

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
```
